# BigRepair
Учебный проект для защиты по окончании курса "Программирование на C#"
Программа генерации отчета о выполненных работах по ремонту.
Для работы с БД - EntityFramework 7.
СУБД - SQLite.
Генерация отчетов - библиотека EPPlusFree.

По умолчанию БД лежит в корне диска D - так прописано в файле appsettings.json.
